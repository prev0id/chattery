# Проработка проекта

В результате проекта должно получиться MVP веб-приложения для группового онлайн общения.

## Функциональные требования

1. Управление пользователем
    1. Регистрация: логин+пароль, проверка уникальности логина, валидация пароля
    2. Авторизация: вход логин+пароль, запоминание сессии, выход из системы
    3. Управление профилем: изменение аватара, ника, возможность смены пароля
2. Чат
    1. Обмен текстовыми сообщениями в реальном времени
    2. Сообщения доставляются всем участникам чата без перезагрузки страницы
    3. Сообщения сохраняются в БД, пользователям доступна история чата
4. Групповые звонки
    1. Поддержка >= 2 участников
    2. Отображение списка участников
    3. Возможность голосового и видео общения

## Не функциональные требования

1. Производительность
    1. Система должна выдерживать не менее 10 одновременных пользователей в чате и звонке
2. Наблюдаемость
    1. Логирование ошибок
    2. Сбор метрик ошибок, времени ответа, активных соединений
3. Безопасность
    1. Хеширование пользовательских данных
4. UX
    1. Простая регистрация
    2. Интуитивный интерфейс, поиск и подключение к чатам, групповые звонки
